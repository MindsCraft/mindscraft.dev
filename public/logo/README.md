# Logo Assets

This folder contains all Mindscraft logo assets in various formats and sizes.

## Folder Structure

```
public/logo/
├── svg/           # Vector formats (scalable)
├── png/           # Raster formats (fixed sizes)
├── favicon/       # Favicon and app icons
└── brand/         # Brand guidelines and usage examples
```

## File Naming Convention

- `logo-full-*` - Complete logo with icon + text
- `logo-icon-*` - Icon only
- `logo-text-*` - Text only  
- `logo-horizontal-*` - Compact horizontal layout
- `*-dark` - For light backgrounds
- `*-light` - For dark backgrounds

## Usage Guidelines

1. **SVG files** - Use for web, scalable applications
2. **PNG files** - Use for presentations, documents, social media
3. **Favicon files** - Use for browser icons and app icons

## Adding New Assets

When adding new logo files:
1. Follow the naming convention
2. Include both dark and light variants when applicable
3. Update the main Logo component if needed
4. Test across different backgrounds and sizes
